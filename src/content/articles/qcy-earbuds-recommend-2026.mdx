**推荐理由：**

如果你**戴不了入耳式**，AilyBuds+ 就是 QCY 产品线里最适合你的选择。

半入耳的设计决定了它没有任何硅胶套塞入耳道的胀痛感，戴一整个下午开会、听音乐、打语音，耳朵都不会抗议。8mm 动圈单元配合定制调音，三频均衡，比很多同价位半入耳的声音更饱满。

**AilyBuds+ 真正的杀招是续航**——单次 8 小时，带盒 32 小时，这个续航数据放在整个 TWS 市场都是第一梯队的。而且它还支持无线充电，119 块的耳机给你配无线充电，这在两年前是想都不敢想的事。

| 项目 | 参数 |
|------|------|
| 类型 | 开放式耳挂式 |
| 降噪 | ❌ 无 |
| 续航 | 6h / 24h（带盒） |
| 防水 | IPX5 |
| 特色 | 耳挂式 360° 环绕音腔 |
| 价格 | ¥139 左右 |

**推荐理由：**

开放式耳机是这两年的风口，QCY 也没落下——ArcBuds 就是他们在开放式赛道的代表作。

**ArcBuds 采用耳挂式设计**，不像骨传导那样需要在颅骨上贴着，也不像入耳式那样塞进耳道。它是两个小喇叭挂在耳朵旁边，把声音定向传到耳道口。好处很明显——**跑步、健身时能听到周围环境声**，安全性拉满；而且不塞耳朵，戴久了完全不会胀痛。

**音质上不能和入耳式比**，开放式形态的天花板就在那里。但 QCY 用了定向传音技术，声音的指向性控制得不错，不会像几十块的开放式耳机那样"外放给旁边的人听"。听播客、打电话、跑步听歌都 OK，拿来认真听交响乐就算了。

**适合运动场景**——IPX5 防水防汗，耳挂夹力适中，跑步不掉，摇头晃脑也不会松动。对运动党来说，ArcBuds 是百元价位最好的开放式耳机之一。

**最适合谁：** 跑步健身党、需要保持环境感知的户外使用者、戴不了入耳式的人。

| 项目 | 参数 |
|------|------|
| 类型 | 入耳式 TWS（游戏向） |
| 降噪 | ❌ 无 |
| 延迟 | **45ms**（游戏模式） |
| 续航 | 6h / 24h（带盒） |
| 特色 | LED 呼吸灯 + 低延迟模式 |
| 价格 | ¥99 左右 |

**推荐理由：**

GTR2 是 QCY 专门为**手游玩家**打造的 TWS，¥99 的价格配上 45ms 超低延迟和 LED 灯效，精准掏空了学生党的钱包。

**低延迟是 GTR2 的核心竞争力**。QCY 的 GTR 系列一直主打游戏场景，GTR2 在游戏模式下能做到 45ms 的延迟——这个数字基本是耳感不可感知的水平。玩《和平精英》听脚步声、玩《王者荣耀》开团听技能音效，声画同步没什么问题。对比之下，普通 TWS 80-150ms 的延迟在 FPS 游戏里就有点难受了。

**LED 呼吸灯**是 GTR2 的一个小加分项（也可能有人觉得土）——充电仓和耳机上都有 RGB 灯效，有几种模式可调，打游戏时确实有那么点电竞氛围感。

**音质方面中规中矩**，毕竟 99 块的耳机、主打游戏，不能指望它有 Melobuds 的听歌体验。但动次打次的低频调音配合游戏音效，反而有点相得益彰——爆炸声、枪声都很有劲儿。

**最适合谁：** 手游玩家、学生党、百元预算只求能响但想有点特色的用户。
