> 如果说 HiFi 圈子里哪个品牌最会把耳机做成艺术品，QoA 必须拥有姓名。这个以鸡尾酒为灵感命名的品牌，用"每一副耳机都是一杯调酒"的态度，在 HIFI 圈杀出了一条独特的路。

QoA（全称 Queen of Audio）是 Kinera 旗下的子品牌，继承了 Kinera 在高端耳机领域的技术积淀，但走出了一条更年轻、更具艺术感的路线。从命名到面板设计再到调音风格，全部围绕"鸡尾酒"这一主题展开——Martini 的优雅、Gin 的清爽、Aviation 的飞行梦……**每款产品都有自己的性格，就像不同的酒，适合不同的心情。**

**品牌调性一句话概括：用调酒师的态度做耳机，好看又好听。**

## 1️⃣ QoA Martini — 旗舰标杆，鸡尾酒之王

| 项目 | 参数 |
|------|------|
| 类型 | 入耳式 HiFi 耳塞 |
| 单元 | 1DD + 4BA + 1BC（7单元混合） |
| 阻抗 | 32Ω |
| 灵敏度 | 110dB |
| 线材 | 高纯度铜银混编 |
| 价格 | ¥4,000+ |

**推荐理由：**

Martini 是 QoA 当下的旗舰型号，名字源自调酒界的"鸡尾酒之王"。作为品牌的声学集大成者，它采用了**1 动圈 + 4 动铁 + 1 骨传导**的豪华七单元配置，从硬件上就奠定了旗舰地位。

**声音方面**，Martini 的声场是 QoA 家族中最开阔的。低频由 10mm 复合振膜动圈负责，下潜深、弹性好，听大编制交响乐时定音鼓的质感非常真实。四颗动铁分别负责中高频和极高频，解析力极强——录音室里原本藏在背景里的三角铁、沙锤细节，Martini 能给你清清楚楚地拎出来。

最妙的是那颗**骨传导单元**，它不负责直接发声，而是通过颧骨传导微振动，让声音的包围感和空间感提升了一个档次。听演唱会现场录音时，你能"感觉"到观众席的声浪，这种沉浸感是普通多单元耳机做不到的。

**面板设计也是艺术品级别**——每一对 Martini 的面板都是手工绘制的，以马提尼酒杯和橄榄为灵感，配合珠光漆面，在不同光线下呈现不同的色泽。拿在手里就是件工艺品。

**最适合谁：** HiFi 发烧友、追求旗舰体验的玩家、对耳机颜值有很高要求的人。

## 2️⃣ QOA Aviation — 飞行主题，中坚实力派

| 项目 | 参数 |
|------|------|
| 类型 | 入耳式 HiFi 耳塞 |
| 单元 | 3BA + 1DD（4单元混合） |
| 阻抗 | 32Ω |
| 灵敏度 | 108dB |
| 价格 | ¥1,300+ |

**推荐理由：**

Aviation 是 QoA 产品线中的"中流砥柱"，也是我个人认为**性价比最高的一款**。

它的名字和设计灵感来自人类航空史——从莱特兄弟的首次飞行到现代喷气机，面板上的纹路仿佛飞机舷窗外的流云。这是一种取悦自己的设计，戴上它仿佛能听到引擎的轰鸣和云端的风声。

**单元配置上**，Aviation 采用 1 颗 10mm 镀钛振膜动圈负责低频，3 颗定制动铁覆盖中高频。整体调音偏向**温暖自然**，中频人声非常毒——听陈奕迅、Adele 这类注重情感表达的歌手的作品时，那种嗓音的颗粒感和气息转折都表现得丝丝入扣。

低频量感适中，不轰头但弹性十足，听流行、摇滚、R&B 都非常对味。高频延伸自然，没有刻意拔亮，久听不累。

**最大的亮点是佩戴舒适度**——腔体设计经过人体工学优化，贴合耳廓，即使长时间聆听也不会感到胀痛。这对于喜欢用耳机专注工作或学习的人来说是个大加分项。

**最适合谁：** 中端 HiFi 入门玩家、人声爱好者、需要长时间佩戴听音的人。

## 3️⃣ QOA Adonis New — 三元素平衡，美学新作

| 项目 | 参数 |
|------|------|
| 类型 | 入耳式 HiFi 耳塞 |
| 单元 | 1DD + 2BA（3单元混合） |
| 阻抗 | 32Ω |
| 灵敏度 | 107dB |
| 价格 | ¥1,200+ |

**推荐理由：**

Adonis New 是 QoA 在经典款 Adonis 基础上的全面升级版。Adonis 在希腊神话中是"美少年"的名字，这款耳机的面板设计也确实配得上这个名号——**天然珍珠贝母+稳定木+树脂**的三重材质融合，每一对都是独一无二的纹理。

**声音方面**，Adonis New 主打"三元素平衡"架构。1 颗 10mm 双磁路动圈负责低频，2 颗 Knowles 定制动铁负责中高频。调音方向比 Aviation 更均衡中性，低频量感恰到好处，中频平直自然，高频顺滑细腻。

它是那种**不想折腾调音的话可以直接退烧**的声音——不偏不倚，各种类型的音乐都能伺候好。古典、爵士、流行、电子都能给出令人满意的表现。对于刚入 HiFi 门又不想走弯路的人来说，Adonis New 是很好的起点。

**线材方面也很有诚意**——标配高纯度无氧铜镀银线，插针是标准的 0.78mm 2pin，方便后续升级换线。原装耳套也提供了多种尺寸和材质选择（硅胶+海绵），开箱体验很到位。

**最适合谁：** 初入 HiFi 的新手、喜欢均衡调音、对耳机颜值有要求的人。

## 4️⃣ QoA Gin — 入门之选，微醺好声音

| 项目 | 参数 |
|------|------|
| 类型 | 入耳式 HiFi 耳塞 |
| 单元 | 1DD（单动圈） |
| 阻抗 | 32Ω |
| 灵敏度 | 105dB |
| 价格 | ¥500+ |

**推荐理由：**

Gin 是 QoA 的入门型号，也是你进入 QoA 世界的**最低门槛**。

名字取自金酒（Gin），设计灵感来自"蓝调之夜"——面板是深邃的午夜蓝搭配金色闪粉，像东京夜景里的霓虹倒映在酒杯中。虽然价格亲民，但设计语言一点没缩水。

**单动圈的配置在千元内很常见，但 QoA 的调音功底让 Gin 的表现远超预期**。10mm 镀钛振膜动圈，带来了这个价位段难得一见的宽松听感。低频蓬松有弹性，中频人声温暖自然，高频柔和不刺。整体风格偏向流行人声，听周杰伦、Taylor Swift 这类音乐非常有味道。

作为入门款，Gin 最大的意义是**让你花最少的钱体验 QoA 的调音哲学**。它不追求极致解析，不追求夸张声场，而是用舒服、耐听的声音告诉你——原来好听的耳机不一定要花大价钱。

**一个小建议**：配合入门的 USB 小尾巴（比如 JCally JM6 或 Moondrop Dawn Pro），Gin 的声音还能再上一个台阶。

**最适合谁：** HiFi 入门探索者、预算有限的学生党、想体验 QoA 调音的入门用户。

## 最后的建议

- **预算无上限、追求旗舰体验** → QoA Martini（骨传导加成，声场一绝）
- **千元级最佳选择** → QOA Aviation（人声毒、佩戴舒适、性价比之王）
- **均衡杂食、不想折腾** → QOA Adonis New（三元素平衡，一耳朵的舒适）
- **先试试水、预算有限** → QoA Gin（最便宜的 QoA 体验卡）

QoA 是一个很有"性格"的品牌。它不像一些大厂那样追求参数上的极致，而是更在意"好听"和"好看"的结合。每款产品都有自己的故事和声音个性，就像不同风味的鸡尾酒——**找到适合自己的那杯，就是最好的。**

如果你是那种"不仅在意耳机声音，还在意它传达的情绪和态度"的人，QoA 大概率会对你的胃口。
