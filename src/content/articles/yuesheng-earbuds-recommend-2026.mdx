**推荐理由：**

如果你对音质的追求已经到了「不想妥协」的程度，天穹 Pro 就是粤声交出的答卷。

这套**一圈一铁混合架构**在千元 TWS 里并不多见——动圈负责低频的松弛和下潜，动铁负责中高频的细节和延展，两者通过粤声自研的数字分频算法衔接，几乎没有可闻的衔接断层。听古典乐时，大提琴的共鸣感（动圈功劳）之后跟上小提琴的泛音细节（动铁优势），层次感非常分明。

**LDAC + aptX Lossless 双高解析协议**的支持意味着，配合安卓手机的开发者模式，你可以跑满接近 990kbps 的码率。虽然听流媒体时差距没那么大，但听本地 FLAC 或 Hi-Res 资源时，细节丰富度明显甩开普通 AAC/TWS+ 编码一大截。

降噪方面，达到一线 TWS 的中上水准——地铁通勤够用，但不是它的核心卖点。毕竟粤声很清楚，买他家产品的人，耳朵里最重要的是音乐本身。

**最适合谁：** HiFi 发烧友、对无线音质有极致追求的人、愿意为音质牺牲一部分降噪体验的用户。

## 2️⃣ 粤声·墨韵 — 纯动铁，人声毒物

| 项目 | 参数 |
|------|------|
| 类型 | 入耳式 TWS |
| 单元 | 定制单动铁 |
| 降噪 | ❌ 物理隔噪 |
| 续航 | 8h / 32h（带盒） |
| 蓝牙 | 蓝牙 5.3，支持 LDAC |
| 防水 | IPX4 |
| 价格 | ¥799 |

**推荐理由：**

这是粤声最有「性格」的一款产品——没有降噪、没有花哨功能，甚至没有混合单元，但它的声音会让你一听就忘不掉。

**单动铁架构**在 TWS 里极少见，因为动铁单元对腔体设计和后端电路要求极高，做不好就是干冷刺。但粤声做到了。墨韵的调音明显偏向中频——人声饱满、结像清晰、口型感极强。听蔡琴《渡口》、听林志炫的现场录音，那种「人声就在你面前唱歌」的临场感，是这个价位其他 TWS 做不到的。

**低频量感不算多**，但质感不错，点到为止。适合听人声、爵士、室内乐的爱好者。如果你听电子乐、摇滚比较多，那墨韵可能偏清淡了，建议看上面的天穹 Pro。

**没有主动降噪**这个点很重要——墨韵靠的是深插入耳式的物理隔音，隔音效果其实不差，办公室和家里够用，但在地铁里就得调高音量了。

**最适合谁：** 人声控、喜欢听女声和爵士的人、能接受无降噪但追求纯净音质的用户。

当 HiFi 品牌做开放式耳机会是什么样？云起是粤声给市场的回答。

**14.2mm 大尺寸复合振膜**配合特殊的导声结构，在开放式形态下做出了让人惊喜的声场表现——横向声场宽广，听 OST 和 Live 录音时能感受到明显的「空间感」。相比同价位开放式耳机常见的"薄声"问题，云起的中低频厚度保持得不错，听感上更接近传统头戴式耳机的自然感。

**开放式天然优势**是佩戴舒适度和长时间续航——10 小时的单次续航在 TWS 里是顶级的，适合全天佩戴。而且不堵塞耳道，跑步、骑行时能听到环境音，更加安全。

**但要注意**，开放式意味着几乎没有隔音，在嘈杂环境下基本听不清。这是物理局限，不是产品问题。
